## Common Questions

### Q: What if we haven't set up budgets?
**A**: This report requires budgets to be entered in the system. Contact your Admin to set up budgets.

### Q: Should I worry about small variances?
**A**: Not usually. Focus on:
- Large dollar variances (> $500)
- Large percentage variances (> 15%)
- Consistent patterns over multiple months

### Q: What causes variances?
**A**: Common reasons:
- **Income**: Seasonal giving, special offerings, declining attendance
- **Expenses**: Emergency repairs, one-time costs, price increases
- **Both**: Budget was unrealistic, circumstances changed

### Q: Can I adjust the budget mid-year?
**A**: Yes, Admin can update budgets. Common at mid-year review.

### Q: What if entire category is way off?
**A**: Investigate thoroughly:
- Review all transactions in that category
- Check if budget was realistic
- Look for one-time events
- Plan adjustments for next period

### Q: How often should I review this?
**A**: 
- **Monthly**: Admin, Bookkeeper
- **Quarterly**: Board review
- **Mid-year**: Comprehensive review with adjustments

## Troubleshooting

### No Budget Data Showing
**Problem**: All budget columns show $0  
**Cause**: Budgets not set up  
**Solution**: Contact Admin to enter budgets

### Percentages Show "N/A"
**Problem**: Can't calculate percentage  
**Cause**: Budget for that account is $0  
**Solution**: Normal if account not budgeted

### Variance Doesn't Match Calculation
**Problem**: Numbers don't add up  
**Cause**: System error or rounding  
**Solution**: Refresh page, recalculate manually

### Can't Filter by Fund
**Problem**: Fund filter not working  
**Solution**: Verify budgets entered per fund
